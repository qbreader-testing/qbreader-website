    GET qbreader.org/api/packet-tossups

    Get only tossups from a packet.

    Parameters
    • setName: string required
      The name of the set. See /set-list for a list of all sets.
    • packetNumber: number required
      The number of the packet in the set, starting from 1.

    Returns
    A JSON object with the following properties:
    • tossups: Tossup[]
      An array of Tossups in the packet.
